MathDuel
=========

Краткое описание
-----------------
MathDuel — простая соревновательная игра для двух игроков, в которой участники решают математические примеры и соревнуются за большее количество правильных ответов. Игра генерирует вопросы разных уровней сложности и отображает варианты ответов; игроки выбирают ответ быстрее соперника.

Технологии
----------
- Jetpack Compose (UI)
- Material Design 3 (Material3)
- Koin (Dependency Injection)
- Kotlin Coroutines

Основная идея
-------------
Приложение предоставляет живую парную механику: два игрока видят один и тот же вопрос и должны как можно быстрее выбрать правильный вариант. Вопросы генерируются динамически и могут иметь разные уровни сложности (Easy / Medium / Hard). После выбора правильного ответа игрок получает очки, игра продолжается до заранее заданного числа раундов.

Ключевые компоненты
-------------------
- `MainActivity` — точка входа, собирает отображение приложения и передаёт зависимости в ViewModel.
- `StartGameScreen` — экран ввода имён игроков и выбора сложности.
- `GameScreen` — основной игровой интерфейс, показывает вопрос и панели игроков.
- `GameViewModel` — управляет состоянием игры, получает вопросы и обрабатывает ответы.
- `GameRepositoryImpl` — локальный генератор вопросов (логика генерации и вариантов ответов).

Быстрый запуск
--------------
1. Откройте проект в Android Studio.
2. Соберите и запустите приложение на эмуляторе или устройстве.
   - В командной строке из корня проекта: `./gradlew assembleDebug` (Windows: `gradlew.bat assembleDebug`).

Примечания
---------
- Генерация вопросов реализована локально в репозитории; при необходимости можно заменить на сетевой источник.
- Логика сложности и задержки подсветки ответов реализована в UI и ViewModel; при желании можно перенести дополнительную логику в слой домена.
